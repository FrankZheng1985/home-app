# 🎨 家庭小助手 - 设计系统

## 📱 小程序尺寸
- 设计稿尺寸: **375 x 812** (iPhone X 标准)
- 状态栏高度: 44px
- TabBar 高度: 83px (含安全区域)
- 安全区域底部: 34px

## 🖼️ 图标设计

### TabBar 图标 (24x24)
需要设计 **线框** 和 **填充** 两种状态：
- 首页: 房子图标
- 家务: 清单/勾选图标
- 动态: 聊天气泡图标
- 我的: 人物图标

### 功能图标 (20x20)
- 设置齿轮
- 编辑铅笔
- 删除垃圾桶
- 分享箭头
- 点赞爱心
- 评论气泡
- 更多三点
- 返回箭头
- 关闭叉号

### 家务类别图标 (32x32)
使用 emoji 或自定义图标：
- 🧹 清洁
- 🍳 厨房
- 🧺 洗涤
- 📦 整理
- 👶 照顾
- ⭐ 其他

## 🎭 动效建议

### 页面转场
- 左右滑动切换页面
- 弹窗从底部滑入

### 微交互
- 按钮点击缩放 (scale: 0.95)
- 点赞爱心跳动动画
- 完成打勾动画
- 列表加载骨架屏

### 动画时长
- 快速: 150ms (按钮反馈)
- 标准: 300ms (页面转场)
- 慢速: 500ms (强调动画)

## 📤 导出规范

### 图片格式
- 图标: SVG (首选) 或 PNG @2x @3x
- 图片: WebP (首选) 或 JPG (质量 80%)
- Logo: SVG + PNG @2x

### 切图命名
```
icon_[name]_[state].png
例如:
icon_home_default.png
icon_home_active.png
```
